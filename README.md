# TakwiniBot – Plateforme intelligente de recrutement

## 📌 Overview
TakwiniBot est une plateforme web intelligente développée dans le cadre du module **Projet Technologies Web (2A)** à **Esprit School of Engineering – Tunisia (2025–2026)**.

Elle permet de mettre en relation les jeunes diplômés tunisiens avec les entreprises grâce à un système de **matching intelligent basé sur l’IA**, ainsi qu’un accompagnement via un **CV intelligent** et des recommandations personnalisées.

---

## 🚀 Features

### 👤 Gestion des utilisateurs
- Création et gestion des comptes utilisateurs
- Gestion des profils
- Vérification de profil
- Suggestions d’amélioration (IA)

### 🏢 Gestion des entreprises
- Ajout et gestion des entreprises
- Publication des offres d’emploi
- Analyse de l’attractivité des offres (IA)

### 📄 Gestion des candidatures
- Postuler à des offres
- Suivi du statut des candidatures
- Classement intelligent des candidats (IA)

### 🤖 Matching intelligent
- Calcul du score de compatibilité (%)
- Recommandation automatique d’offres
- Génération et gestion du matching

### 📑 CV intelligent
- Création et gestion de CV
- Analyse automatique du CV
- Suggestions d’amélioration (IA)

### 🌱 Score RSE
- Calcul du score RSE des entreprises
- Classement des entreprises
- Filtrage selon critères durables

## 🧱 Architecture
Le projet est structuré en plusieurs modules :
- Gestion des utilisateurs
- Gestion des entreprises
- Gestion des candidatures
- Matching intelligent
- CV intelligent
- Score RSE

Chaque module est connecté via des relations entre entités :
- User ↔ Profil
- Entreprise ↔ Offre
- User ↔ Candidature ↔ Offre
- User ↔ Matching ↔ Offre
- User ↔ CV
- Entreprise ↔ ScoreRSE
